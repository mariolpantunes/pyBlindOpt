pyBlindOpt.fa

Firefly Algorithm (FA) optimization.

This module implements the Firefly Algorithm, inspired by the flashing behavior of fireflies. Fireflies are attracted to each other based on brightness (fitness), but light intensity decreases with distance.

Analogy: Fireflies move towards brighter peers. If no one is brighter, they move randomly. The attractiveness depends on distance, simulating light absorption in air.

Mathematical Formulation: The movement of firefly $i$ towards $j$ is: $$ x_i^{t+1} = x_i^t + \beta_0 e^{-\gamma r_{ij}^2} (x_j^t - x_i^t) + \alpha (rand - 0.5) $$ where $r_{ij}$ is the distance, $\beta_0$ is attractiveness at $r=0$, and $\alpha$ is the randomization parameter.

 82        """
 83        Updates the global best firefly.
 84

 87        """
 88        best_idx = np.argmin(self.scores)

 91            self.best_pos = self.pop[best_idx].copy()

 93    def _generate_offspring(self, epoch: int) -> np.ndarray:
 94        """
 95        Calculates firefly movements.
 96
 97        Computes the pairwise attraction vector sum for every firefly $i$ towards all brighter fireflies $j$.
 98
 99        **Attractiveness:**
100        $$ \\beta = \\beta_0 e^{-\\gamma r^2} $$
101        **Movement:**
102        $$ x_{new} = x_{old} + \\sum_{j \\in \text{better}} \\beta_{ij}(x_j - x_i) + \\text{randomness} $$
103
104        Args:
105            epoch (int): Current iteration.
106
107        Returns:
108            np.ndarray: The new positions of the fireflies.
109        """
110        # 1. Compute pairwise distances (N, N)
111        # using utils.global_distances or manual broadcasting
112        # diff[i, j, d] = pop[j, d] - pop[i, d]
113        pop = self.pop
114        diff = pop[:, np.newaxis, :] - pop[np.newaxis, :, :]
115        r_sq = np.sum(diff**2, axis=-1)  # (N, N) distance squared
116
117        # 2. Compute Attractiveness Beta (N, N)
118        # beta[i, j] = beta0 * exp(-gamma * r^2)
119        beta = self.beta0 * np.exp(-self.gamma * r_sq)
120
121        # 3. Create Mask: Move i towards j ONLY IF score[j] < score[i]
122        # mask[i, j] is True if j is better than i
123        scores_flat = self.scores.flatten()
124        mask = scores_flat[np.newaxis, :] < scores_flat[:, np.newaxis]
125
126        # Zero out beta where j is not better than i
127        beta_masked = (beta * mask)[:, :, np.newaxis]
128
129        # 4. Compute Movement Steps
130        # Sum over j: sum(beta * (x_j - x_i))
131        # tensordot or sum(..., axis=1)
132        # movement[i] = sum_j ( beta_masked[i, j] * diff[i, j] )
133        movement = np.sum(beta_masked * diff, axis=1)
134
135        # 5. Randomness
136        bound_width = self.bounds[:, 1] - self.bounds[:, 0]
137        random_step = self.alpha * (self.rng.random(pop.shape) - 0.5) * bound_width
138
139        # 6. Apply
140        new_pop = pop + movement + random_step
141
142        return new_pop
143
144    def _selection(self, offspring: np.ndarray, offspring_scores: np.ndarray):
145        """
146        Greedy Selection.
147
148        Updates the population only if the new position provides a better objective value.
149
150        Args:
151            offspring (np.ndarray): Candidate positions.
152            offspring_scores (np.ndarray): Candidate scores.
153        """
154        improved_mask = offspring_scores < self.scores
155        self.pop[improved_mask] = offspring[improved_mask]
156        self.scores[improved_mask] = offspring_scores[improved_mask]

pyBlindOpt

pyBlindOpt is a library that implements several derivative-free optimization algorithms (including genetic and evolutionary methods).

Currently, it implements thirteen different algorithms:

  1. Random Search (RS): A baseline optimization method that iteratively generates candidate solutions from the search space according to a specified probability distribution (usually uniform) and records the best solution found.
  2. Hill Climbing (HC): A local search algorithm that starts with an arbitrary solution and attempts to find a better one by making incremental changes (Greedy approach).
  3. Simulated Annealing (SA): A probabilistic technique designed to escape local optima by allowing "uphill" moves (worse solutions) with a probability that decreases over time (simulating the cooling process of metallurgy).
  4. Genetic Algorithm (GA): An evolutionary algorithm inspired by natural selection. It generates high-quality solutions using biologically inspired operators such as mutation, crossover, and selection.
  5. Differential Evolution (DE): A population-based method that optimizes a problem by iteratively improving a candidate solution using the differences between randomly selected vectors.
  6. Particle Swarm Optimization (PSO): A metaheuristic where particles move through the search space guided by their own best-known position and the swarm's global best-known position.
  7. Grey Wolf Optimization (GWO): A population-based algorithm that simulates the leadership hierarchy (Alpha, Beta, Delta, Omega) and hunting mechanism of grey wolves.
  8. Enhanced Grey Wolf Optimization (EGWO): An advanced variant of GWO that incorporates a weighted prey position and stochastic error to better balance exploration and exploitation.
  9. Artificial Bee Colony (ABC): Simulates the foraging behavior of honey bees, utilizing employed, onlooker, and scout bees to exploit food sources and explore new ones.
  10. Firefly Algorithm (FA): Inspired by the flashing behavior of fireflies, where attraction is proportional to brightness (fitness) and decreases with distance.
  11. Harris Hawks Optimization (HHO): Mimics the cooperative hunting behavior of Harris' hawks, featuring distinct exploration and exploitation phases controlled by the prey's escaping energy.
  12. Cuckoo Search (CS): Based on the brood parasitism of cuckoos, utilizing Lévy flights for global exploration and nest abandonment for local optima avoidance.
  13. Honey Badger Algorithm (HBA): Mimics the intelligent foraging behavior of honey badgers, switching between digging (using smell intensity) and following honeyguides.

All algorithms take advantage of the joblib library to parallelize objective function evaluations and cache results for improved performance.

Note: The code has been optimized to a certain degree but was primarily created for educational purposes. Please consider libraries like pymoo or SciPy if you require a production-grade implementation.

107        """
108        Generates candidate solutions via Gaussian perturbation.
109
110        Creates a neighbor solution $x_{new}$ by adding normally distributed noise
111        to the current solution $x_{curr}$:
112        $$ x_{new} = x_{curr} + \\mathcal{N}(0, \\sigma^2) $$
113        where $\\sigma$ is the `step_size`.
114
115        Args:
116            epoch (int): The current iteration index.
117
118        Returns:
119            np.ndarray: The candidate solutions (offspring).
120        """
121        noise = self.rng.normal(loc=0.0, scale=self.step_size, size=self.pop.shape)
122        return self.pop + noise

124    def _selection(self, offspring: np.ndarray, offspring_scores: np.ndarray):
125        """
126        Greedy Selection (Local Search).
127
128        Accepts a new solution only if it is strictly better than the current one.
129
130        $$
131        x_{t+1} = \\text{argmin}(f(x_t), f(x_{new}))
132        $$
133

136            offspring_scores (np.ndarray): The objective values of the candidates.
137        """
138        improved_mask = offspring_scores < self.scores
139
140        self.pop[improved_mask] = offspring[improved_mask]
141        self.scores[improved_mask] = offspring_scores[improved_mask]

 30class SimulatedAnnealing(Optimizer):
 31    """
 32    Simulated Annealing Optimizer.
 33
 34    A probabilistic local search algorithm that approximates global optimization
 35    by accepting worse solutions with a probability that decreases over time (Temperature).
 36
 37    If n_pop > 1, this acts as 'Parallel Simulated Annealing', maintaining 'n_pop'
 38    independent annealing chains.
 39
 40    **Cooling Schedule:**
 41    Fast Annealing (Cauchy):
 42    $$ T_k = \\frac{T_0}{k + 1} $$
 43    """
 44
 45    @utils.inherit_signature(Optimizer)
 46    def __init__(
 47        self,
 48        objective: collections.abc.Callable,
 49        bounds: np.ndarray,
 50        step_size: float = 0.01,
 51        temp: float = 20.0,
 52        **kwargs,
 53    ):
 54        self.step_size = step_size
 55        self.initial_temp = temp
 56        self.current_temp = temp
 57
 58        super().__init__(objective=objective, bounds=bounds, **kwargs)
 59
 60    def _initialize(self):
 61        """
 62        Resets temperature to initial value.
 63        """
 64        self.current_temp = self.initial_temp
 65
 66    def _update_iter_params(self, epoch: int):
 67        """
 68        Updates temperature $T$.
 69        $$ T_{new} = T_0 / (epoch + 1) $$
 70        """
 71        self.current_temp = self.initial_temp / float(epoch + 1)
 72
 73    def _update_best(self, epoch: int):
 74        """
 75        Updates global best from current population chains.
 76        """
 77        best_idx = np.argmin(self.scores)
 78        if self.scores[best_idx] < self.best_score:
 79            self.best_score = self.scores[best_idx]
 80            self.best_pos = self.pop[best_idx].copy()
 81
 82    def _generate_offspring(self, epoch: int) -> np.ndarray:
 83        """
 84        Gaussian perturbation.
 85        $$ x_{new} = x_{curr} + \\mathcal{N}(0, \\sigma^2) $$
 86        """
 87        noise = self.rng.normal(loc=0.0, scale=self.step_size, size=self.pop.shape)
 88        return self.pop + noise
 89
 90    def _selection(self, offspring: np.ndarray, offspring_scores: np.ndarray):
 91        """
 92        Metropolis Criterion.
 93
 94        $$ P(\\text{accept}) = \\exp\\left(-\\frac{\\Delta E}{T}\\right) $$
 95        Accepts if Improved ($\\Delta E < 0$) OR if Random < $P(\\text{accept})$.
 96        """
 97        # Calculate Delta E (Candidate - Current)
 98        # Note: Optimization implies Minimization.
 99        diff = offspring_scores - self.scores
100
101        # 1. Check for strict improvement
102        # Mask where candidate is better (diff < 0)
103        improve_mask = diff < 0
104
105        # 2. Check for Metropolis acceptance (for worse solutions)
106        # P = exp(-diff / T)
107        # We only compute this where diff >= 0 to avoid warnings/unnecessary work
108        metropolis_mask = np.zeros(self.n_pop, dtype=bool)
109
110        worse_indices = ~improve_mask
111        if np.any(worse_indices):
112            # Probability of accepting worse solution
113            # Ensure T is not zero to avoid division by zero error
114            safe_temp = max(self.current_temp, 1e-10)
115            acceptance_prob = np.exp(-diff[worse_indices] / safe_temp)
116
117            # Generate random numbers [0, 1] for these indices
118            count = int(np.sum(worse_indices))
119            rand_vals = self.rng.random(size=count)
120
121            # Accept if Random < Prob
122            accepted_worse = rand_vals < acceptance_prob
123
124            # Map back to full mask
125            metropolis_mask[worse_indices] = accepted_worse
126
127        # Combine: Accept if (Improved OR Metropolis)
128        accept_mask = improve_mask | metropolis_mask
129
130        # Update population
131        self.pop[accept_mask] = offspring[accept_mask]
132        self.scores[accept_mask] = offspring_scores[accept_mask]

310        """

321    def _generate_offspring(self, epoch: int) -> np.ndarray:
322        """
323        Executes the GA Loop.
324
325        1.  **Selection:** Creates a mating pool of size $N$.
326        2.  **Crossover:** Pairs parents and produces children.
327        3.  **Mutation:** Mutates children.
328
329        Args:
330            epoch (int): Current iteration.
331
332        Returns:
333            np.ndarray: The next generation.
334        """
335        # 1. Selection
336        # Select n_pop parents
337        # Note: We pass self.rng to ensure reproducibility if operators support it
338        selected = []
339        for _ in range(self.n_pop):
340            # Check if operator accepts rng
341            try:
342                s = self.selection_op(self.pop, self.scores, rng=self.rng)
343            except TypeError:
344                s = self.selection_op(self.pop, self.scores)
345            selected.append(s)
346
347        # 2. Crossover & Mutation
348        children = []
349
350        # Work in pairs (0,1), (2,3), etc.
351        # Ensure we don't go out of bounds if n_pop is odd
352        limit = self.n_pop - (self.n_pop % 2)
353
354        for i in range(0, limit, 2):
355            p1, p2 = selected[i], selected[i + 1]
356
357            # Apply Crossover
358            try:
359                offspring_list = self.crossover_op(p1, p2, self.r_cross, rng=self.rng)
360            except TypeError:
361                offspring_list = self.crossover_op(p1, p2, self.r_cross)
362
363            # Apply Mutation to each child
364            for child in offspring_list:
365                # We stop adding if we reached n_pop (e.g., linear crossover produces 3 children)
366                if len(children) >= self.n_pop:
367                    break
368
369                try:
370                    mutant = self.mutation_op(
371                        child, self.r_mut, self.bounds, rng=self.rng
372                    )
373                except TypeError:
374                    mutant = self.mutation_op(child, self.r_mut, self.bounds)
375
376                children.append(mutant)
377
378            if len(children) >= self.n_pop:
379                break
380
381        # 3. Fill remaining spots (if any)
382        # If crossover produced fewer children or n_pop was odd
383        while len(children) < self.n_pop:
384            # Fallback: copy the last selected parent or random
385            children.append(selected[-1].copy())
386
387        return np.array(children)

116    def _mutation(self, candidates: np.ndarray) -> np.ndarray:
117        """
118        Applies Differential Mutation.
119
120        $$ v = x_{base} + F \\sum (x_{rA} - x_{rB}) $$
121
122        Args:
123            candidates (np.ndarray): Array where index 0 is the base vector and subsequent indices are pairs for difference calculation.
124
125        Returns:
126            np.ndarray: The mutant vector.
127        """
128        diff_sum = np.zeros_like(candidates[0])
129
130        # Iterate pairs: (1,2), (3,4), etc.
131        for i in range(1, len(candidates), 2):
132            diff_sum += candidates[i] - candidates[i + 1]
133
134        return candidates[0] + self.F * diff_sum

136    def _crossover_binom(self, target: np.ndarray, mutant: np.ndarray) -> np.ndarray:
137        """
138        Binomial Crossover.
139
140        Each gene is swapped with probability $CR$. Ensures at least one gene is changed.
141
142        Args:
143            target (np.ndarray): The parent vector.
144            mutant (np.ndarray): The donor vector.
145
146        Returns:
147            np.ndarray: The trial vector.
148        """
149        dim = target.shape[0]
150        # 1. Generate random mask
151        mask = self.rng.random(dim) < self.cr
152        # 2. Force at least one index to change (standard DE guarantee)
153        j_rand = self.rng.integers(0, dim)

161        Exponential Crossover.
162
163        Swaps a contiguous block of genes starting from a random index.
164
165        Args:
166            target (np.ndarray): The parent vector.
167            mutant (np.ndarray): The donor vector.
168
169        Returns:
170            np.ndarray: The trial vector.
171        """
172        dim = target.shape[0]
173        trial = target.copy()
174
175        j = self.rng.integers(0, dim)
176        L = 0
177        while self.rng.random() < self.cr and L < dim:
178            trial[j] = mutant[j]
179            j = (j + 1) % dim
180            L += 1

182        return trial

 47        """

 49

 51        $$ \\sigma_t = \\exp(-100 \\frac{t+1}{T}) $$
 52        This ensures high randomness (exploration) at the start and low randomness (exploitation) at the end.
 53

 59    def _generate_offspring(self, epoch: int) -> np.ndarray:
 60        """
 61        Generates new positions using Weighted Prey + Noise.
 62
 63        1.  **Weights:** Randomly generates weights for $\\alpha, \\beta, \\gamma$.
 64        2.  **Target:** Calculates weighted center + Gaussian noise.
 65        3.  **Update:** Moves wolves towards this target.
 66

 69
 70        Returns:
 71            np.ndarray: The new positions.
 72        """
 73        dim = self.pop.shape[1]
 74
 75        # 1. Calculate Weights (Omega)
 76        # Random uniform [1, 3], normalized, and sorted descending
 77        omega = self.rng.uniform(1, 3, size=3)
 78        omega /= np.sum(omega)
 79        omega = np.sort(omega)[::-1]  # Sort descending (Alpha gets highest weight)
 80

 82        # Weighted sum of leaders + Stochastic Noise
 83        noise = self.rng.normal(0, self.epoch_std, size=dim)
 84

 87            + omega[1] * self.beta_pos
 88            + omega[2] * self.gamma_pos
 89            + noise
 90        )
 91
 92        # 3. Update Population Positions
 93        # X_new = X_old - Uniform(-2, 2) * |Prey - X_old|
 94
 95        # Random factor shape: (N, D)
 96        rand_factor = self.rng.uniform(-2, 2, size=(self.n_pop, dim))
 97
 98        # Broadcasting: prey (D,) - pop (N, D)
 99        distance_to_prey = np.abs(prey - self.pop)

138        # Robust inversion for costs that can be negative:

 99        """
100        Calculates firefly movements.
101
102        Computes the pairwise attraction vector sum for every firefly $i$ towards all brighter fireflies $j$.
103
104        **Attractiveness:**
105        $$ \\beta = \\beta_0 e^{-\\gamma r^2} $$
106        **Movement:**
107        $$ x_{new} = x_{old} + \\sum_{j \\in \text{better}} \\beta_{ij}(x_j - x_i) + \\text{randomness} $$
108
109        Args:
110            epoch (int): Current iteration.
111
112        Returns:
113            np.ndarray: The new positions of the fireflies.
114        """
115        # 1. Compute pairwise distances (N, N)
116        # using utils.global_distances or manual broadcasting
117        # diff[i, j, d] = pop[j, d] - pop[i, d]
118        pop = self.pop
119        diff = pop[:, np.newaxis, :] - pop[np.newaxis, :, :]
120        r_sq = np.sum(diff**2, axis=-1)  # (N, N) distance squared
121
122        # 2. Compute Attractiveness Beta (N, N)
123        # beta[i, j] = beta0 * exp(-gamma * r^2)
124        beta = self.beta0 * np.exp(-self.gamma * r_sq)
125
126        # 3. Create Mask: Move i towards j ONLY IF score[j] < score[i]
127        # mask[i, j] is True if j is better than i
128        scores_flat = self.scores.flatten()
129        mask = scores_flat[np.newaxis, :] < scores_flat[:, np.newaxis]
130
131        # Zero out beta where j is not better than i
132        beta_masked = (beta * mask)[:, :, np.newaxis]
133
134        # 4. Compute Movement Steps
135        # Sum over j: sum(beta * (x_j - x_i))
136        # tensordot or sum(..., axis=1)
137        # movement[i] = sum_j ( beta_masked[i, j] * diff[i, j] )
138        movement = np.sum(beta_masked * diff, axis=1)
139
140        # 5. Randomness
141        bound_width = self.bounds[:, 1] - self.bounds[:, 0]
142        random_step = self.alpha * (self.rng.random(pop.shape) - 0.5) * bound_width
143
144        # 6. Apply
145        new_pop = pop + movement + random_step
146
147        return new_pop

 65    def _update_best(self, epoch: int):
 66        """
 67        Updates the Rabbit position (Global Best).
 68
 69        In HHO, the global best is referred to as the "Rabbit".
 70
 71        Args:
 72            epoch (int): Current iteration.
 73        """
 74        best_idx = np.argmin(self.scores)

 77            self.best_pos = self.pop[best_idx].copy()

 79    def _generate_offspring(self, epoch: int) -> np.ndarray:
 80        """
 81        Generates new positions based on the HHO phases.
 82
 83        **1. Exploration ($|E| \\ge 1$):**
 84        Move based on random hawk or average position.
 85
 86        **2. Exploitation ($|E| < 1$):**
 87        * **Soft Besiege:** $x_{new} = \\text{Rabbit} - E | J \\cdot \\text{Rabbit} - x |$
 88        * **Hard Besiege:** $x_{new} = \\text{Rabbit} - E | \\text{Rabbit} - x |$
 89        * **Soft/Hard with Rapid Dives:** Uses Lévy flights ($LF$) to perform zig-zag movements if the besiege fails ($Z = Y + S \times LF$).
 90
 91        Args:
 92            epoch (int): Current iteration.
 93
 94        Returns:
 95            np.ndarray: The new positions of the hawks.
 96        """
 97        rabbit = self.best_pos
 98        E0 = 2 * self.rng.random(self.n_pop) - 1
 99        E = 2 * E0 * (1 - (epoch / self.n_iter))
100
101        X_new = np.zeros_like(self.pop)
102        mean_hawk = np.mean(self.pop, axis=0)
103
104        # Iterate per hawk (vectorizing HHO's 4 branches is complex and prone to bugs)
105        for i in range(self.n_pop):
106            energy = E[i]
107            x = self.pop[i]
108
109            if abs(energy) >= 1:  # Exploration
110                q = self.rng.random()
111                if q >= 0.5:
112                    rand_idx = self.rng.integers(0, self.n_pop)
113                    rand_hawk = self.pop[rand_idx]
114                    r1, r2 = self.rng.random(), self.rng.random()
115                    X_new[i] = rand_hawk - r1 * np.abs(rand_hawk - 2 * r2 * x)
116                else:
117                    r3, r4 = self.rng.random(), self.rng.random()
118                    term = self.bounds[:, 0] + r4 * (
119                        self.bounds[:, 1] - self.bounds[:, 0]
120                    )
121                    X_new[i] = (rabbit - mean_hawk) - r3 * term
122            else:  # Exploitation
123                r = self.rng.random()
124
125                # Soft Besiege
126                if r >= 0.5 and abs(energy) >= 0.5:
127                    J = 2 * (1 - self.rng.random())
128                    X_new[i] = (rabbit - x) - energy * np.abs(J * rabbit - x)
129
130                # Hard Besiege
131                elif r >= 0.5 and abs(energy) < 0.5:
132                    X_new[i] = rabbit - energy * np.abs(rabbit - x)
133
134                # Rapid Dives (Soft & Hard)
135                else:
136                    # Base target Y
137                    if abs(energy) >= 0.5:  # Phase 3 (Soft)
138                        J = 2 * (1 - self.rng.random())
139                        Y = rabbit - energy * np.abs(J * rabbit - x)
140                    else:  # Phase 4 (Hard)
141                        J = 2 * (1 - self.rng.random())
142                        Y = rabbit - energy * np.abs(J * rabbit - mean_hawk)
143
144                    # Dive target Z (Levi Flight)
145                    dim = self.bounds.shape[0]
146                    S = self.rng.random(dim)
147                    levy = utils.levy_flight(1, dim, 1.5, self.rng)[0]
148                    Z = Y + S * levy
149
150                    # Selection internal to offspring generation
151                    # We must evaluate to decide between Y and Z
152                    Y = self._check_bounds(Y[np.newaxis, :])
153                    Z = self._check_bounds(Z[np.newaxis, :])
154
155                    score_Y = self.evaluate(Y)[0]
156                    score_Z = self.evaluate(Z)[0]
157
158                    # Greedy choice between Y, Z (and implicit X_old)
159                    # We return the best of Y or Z.
160                    # The base class _selection will compare this result vs X_old.
161                    if score_Y < score_Z:
162                        X_new[i] = Y[0]
163                    else:
164                        X_new[i] = Z[0]
165
166                    # Note: If both Y and Z are worse than X_old, we still return one here.
167                    # _selection will reject it later.
168
169        return X_new

 75    def _update_best(self, epoch: int):
 76        """
 77        Updates the Prey position (Global Best).
 78
 79        Args:
 80            epoch (int): Current iteration.
 81        """
 82        best_idx = np.argmin(self.scores)
 83        if self.scores[best_idx] < self.best_score:
 84            self.best_score = self.scores[best_idx]
 85            self.best_pos = self.pop[best_idx].copy()
 86
 87    def _generate_offspring(self, epoch: int) -> np.ndarray:
 88        """
 89        Generates new positions based on Digging or Honey phases.
 90
 91        Calculates smell intensity $I$ based on inverse square distance.
 92        Switches between phases probabilistically.
 93
 94        Args:
 95            epoch (int): Current iteration.
 96
 97        Returns:
 98            np.ndarray: New badger positions.
 99        """
100        alpha = self.C * math.exp(-epoch / self.n_iter)
101        prey = self.best_pos
102        new_pop = np.zeros_like(self.pop)
103
104        for i in range(self.n_pop):
105            # Intensity I
106            dist = np.linalg.norm(self.pop[i] - prey)
107            dist_sq = dist * dist if dist > 1e-10 else 1e-10
108            In = self.rng.random() * self.C / (4 * math.pi * dist_sq) * alpha
109
110            # Direction F
111            F = 1 if self.rng.random() <= 0.5 else -1
112
113            if self.rng.random() <= 0.5:
114                r3, r4, r5 = self.rng.random(), self.rng.random(), self.rng.random()
115
116                term_attract = F * self.beta * In * prey
117                term_dist = (
118                    F
119                    * r3
120                    * alpha
121                    * dist
122                    * np.abs(
123                        math.cos(2 * math.pi * r4) * (1 - math.cos(2 * math.pi * r5))
124                    )
125                )
126
127                new_pop[i] = prey + term_attract + term_dist
128            else:  # Honey
129                r7 = self.rng.random()
130                new_pop[i] = prey + F * r7 * alpha * dist
131
132        return new_pop
133
134    def _selection(self, offspring: np.ndarray, offspring_scores: np.ndarray):
135        """
136        Greedy Selection.
137
138        Args:
139            offspring (np.ndarray): New positions.
140            offspring_scores (np.ndarray): New scores.
141        """
142        improved_mask = offspring_scores < self.scores
143        self.pop[improved_mask] = offspring[improved_mask]
144        self.scores[improved_mask] = offspring_scores[improved_mask]
